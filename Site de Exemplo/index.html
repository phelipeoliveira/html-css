Diferenças entre XHTML, HTML 4 e HTML 5

  • A seguir vcs verão um exemplo de um site index.xhtml

      
        <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
        "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
      
        <html xmlns="http://www.w3.org/1999/xhtml">
          <head>
             <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
             <title>Exemplo de XHTML by WikiPedia></title>
          </head>
          <body>
              <p>Olá Mundo, Estou criando um site</p>
          </body>
        </html>

  • Agora vejam um exemplo de um site index.html

      
        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo de HTML 5</title>
        </head>
        <body>
            <h1>Olá Mundo, Estou criando um site</h1>
        </body>
        </html>

  • Veja abaixo um exemplo de site index.html em html 4

    •          
          <html>
            <head>
                <title>Meu site HTML4</title>
            </head>
            <body>
                <center><h1>Olá Mundo, Estou criando um site</h1></center>
            </body>
        </html>

Tanto o HTML 4 quanto o XHTML tem códigos obsoletos, ou seja, que a qualquer hora pode parar de funcionar, então é recomendado usar HTML 5
